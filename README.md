# Markdown-It-Browser-Library

Make for [CUP Online Judge](http://acm.cup.edu.cn)

## Usage
**Browser:**

```html
<script src="/markdown-it.js"></script>

<!-- some code -->
<script>
markdownIt.render(`# Hello MarkdownIt!`)
</script>
```

## Dependencies
* markdown-it
* markdown-it-katex
* markdown-it-highlightjs
* github-markdown.css

## Build

```bash
npm install
webpack
```
